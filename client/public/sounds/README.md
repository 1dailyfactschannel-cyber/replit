# Звуковые файлы для уведомлений

Для работы звуковых уведомлений необходимо добавить следующие файлы в эту директорию:

## Необходимые файлы:

1. **reminder.mp3** - Звук напоминания о предстоящей встрече
   - Длительность: 1-2 секунды
   - Рекомендуется: Короткий приятный звук (ding, chime, etc.)
   - Громкость: 50%

2. **calendar_update.mp3** - Звук при обновлении календаря
   - Длительность: 0.5-1 секунда
   - Рекомендуется: Короткий щелчок или свист
   - Громкость: 50%

## Где взять звуки:

1. **Бесплатные ресурсы:**
   - [Freesound](https://freesound.org/) - Большая библиотека звуков
   - [Mixkit](https://mixkit.co/free-sound-effects/notification/) - Бесплатные звуки уведомлений
   - [Zapsplat](https://www.zapsplat.com/) - Бесплатные звуковые эффекты

2. **Примеры поисковых запросов:**
   - "notification sound"
   - "calendar reminder ding"
   - "soft chime notification"
   - "gentle alert sound"

## Требования:

- Формат: MP3
- Размер: до 100KB каждый
- Кодек: MPEG Audio Layer III
- Битрейт: 128 kbps достаточно

## После добавления файлов:

Звуки будут автоматически воспроизводиться при:
- Напоминаниях о встречах из Яндекс Календаря
- Обновлениях событий
- Новых событиях

## Примечание:

Браузеры могут блокировать автовоспроизведение звуков до первого взаимодействия пользователя со страницей. Это нормальное поведение для безопасности.
